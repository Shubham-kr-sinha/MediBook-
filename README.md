# Smart Medical Appointment System

A MERN stack application for managing medical appointments.

## Tech Stack
- **Frontend**: React (Vite), TailwindCSS
- **Backend**: Node.js, Express, MongoDB (Mongoose)
- **Auth**: JWT, bcryptjs

## Getting Started

### Server
1. Navigate to `server/`.
2. `npm install`
3. Create `.env` (already done).
4. `npm run dev` (Runs on port 5000).

### Client
1. Navigate to `client/`.
2. `npm install`
3. `npm run dev` (Runs on port 5173).

## Features
- **Admin**: Dashboard to add doctors and view stats.
- **Doctor**: Dashboard to view and manage appointments.
- **Patient**: Dashboard to book appointments and view history.
